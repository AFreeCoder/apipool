# 常见问题

---

## API 请求返回 401 错误？

401 错误表示认证失败，请按以下步骤排查：

1. **检查 API Key 是否正确**：确认复制时没有遗漏或多余的字符。
2. **检查 API Key 状态**：进入 `/keys` 页面，确认 Key 处于 **active** 状态，未被禁用或过期。
3. **检查 Header 格式**：确认认证 Header 格式正确，例如 `Authorization: Bearer YOUR_API_KEY`（注意 `Bearer` 后有一个空格）。
4. **确认 Key 未被删除**：如果 Key 已被删除，需要重新创建一个新的 API Key。

---

## API 请求返回 403 错误？

403 错误表示请求被拒绝，可能的原因和解决方法：

1. **余额不足**：登录仪表盘（`/dashboard`）查看账户余额，如余额为零，需要充值或兑换余额卡。
2. **IP 被限制**：检查 API Key 的 IP 白名单/黑名单设置（`/keys` 页面编辑对应 Key），确认你的请求 IP 在允许范围内。
3. **账号被禁用**：如果以上排查均无问题，可能是账号被管理员禁用，请联系平台管理员处理。

---

## API 请求返回 429 错误？

429 错误表示请求频率或用量超限，可能的原因和解决方法：

1. **订阅限额耗尽**：进入 `/subscriptions` 页面查看订阅的日/周/月限额使用情况，如果限额已满，需要等待窗口重置（日限额 24 小时、周限额 7 天、月限额 30 天）。
2. **并发超限**：减少同时发起的请求数量，避免短时间内发送大量并发请求。
3. **上游速率限制**：上游 AI 服务本身有速率限制，稍后重试即可。如果频繁出现，联系管理员了解详情。

---

## 如何查看剩余额度？

根据你的计费模式，有以下几种方式查看额度：

- **账户余额**：登录后在仪表盘（`/dashboard`）查看当前可用余额。
- **订阅限额**：进入"我的订阅"页面（`/subscriptions`），查看各项限额的使用进度条和剩余量。
- **API Key 配额**：在 API Keys 页面（`/keys`）查看每个 Key 的已用配额和总配额。

---

## 支持哪些模型？

可用模型取决于你的 API Key 所在分组配置的账号。通过以下接口可以查询当前可用的模型列表：

```bash
curl https://your-api-domain/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

平台常见的模型包括：

- **Claude 系列**：claude-opus-4-6、claude-sonnet-4-5、claude-haiku-4-5 等
- **Gemini 系列**：gemini-2.5-pro、gemini-2.5-flash 等
- **GPT 系列**：取决于平台配置

具体可用模型以接口返回结果为准。

## 流式响应中断了怎么办？

流式请求有超时限制（默认 600 秒）。如果响应中途中断，请按以下步骤排查：

1. **检查网络连接**：确认网络稳定，尝试切换网络环境后重试。
2. **减少 `max_tokens` 参数**：过大的 `max_tokens` 会导致生成时间过长，可能触发超时，建议适当降低。
3. **重试请求**：偶尔的中断可能是上游服务临时波动，直接重试即可。
4. **联系管理员**：如果频繁中断，请联系平台管理员确认上游服务状态。
