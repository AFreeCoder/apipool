## 一、平台充值方式

1. 左侧菜单 → **充值订阅**，选择商品后点击**购买**
2. 购买完成，获得**兑换码**
3. 左侧菜单 → **兑换**，输入兑换码即完成充值

## 二、平台费率计算

**充值汇率：1 元人民币 = 1 美元**（充多少人民币，余额就增加多少美元）

API 调用费用按**官方价格**计算，每个分组有一个**倍率系数**：

| 分组系数 | 含义 |
|---------|------|
| 1.0 | 与官方价格一致（美元） |
| > 1.0 | 高于官方价格（美元） |
| < 1.0 | 低于官方价格（美元） |

实际平台扣费 = **官方价格 × 分组系数**

> **关于缓存：** 平台已开启 Prompt Cache，长对话中的重复上下文可被缓存，缓存命中后费用大幅降低，即使很长的对话，实际花费也相当可控。

## 三、如何查看自己的花费

有两个地方可以查看花费记录：**仪表盘** 和 **使用记录**

### 仪表盘

仪表盘数据为汇总数据，可查看账户余额、API密钥数、今日消费数据等

### 使用记录

左侧菜单 → **使用记录**

- **实际消费**：按分组系数扣费后的实际花费
- **标准消费**：按官方 API 价格计算的参考值（不含系数）
- 底部可切换统计口径：**按密钥** 或 **按时间**

![](https://tjjsjwhj-blog.oss-cn-beijing.aliyuncs.com/2026/03/05/17726411556683.jpg)


点击某条记录的 **Token** 详情，可查看：

| 类型 | 说明 |
|------|------|
| 输入 Token | 发送给模型的内容 |
| 输出 Token | 模型返回的内容 |
| 写缓存 Token | 首次写入缓存的内容（略高于输入单价） |
| 读缓存 Token | 命中缓存的内容（远低于输入单价） |

以及每种 Token 对应的单价，方便核对费用。
